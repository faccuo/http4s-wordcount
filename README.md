# Word count

Initial iteration on the word count implementation.

## Usage

``` bash
$ sbt run
```

## Testing

``` bash
$ sbt test
```

## Manual load testing

Big and small sample files (~10mb and ~250kb) have been provided. Using Apache Bench for simple load testing:

``` bash
$ ab -n 100 -c 5 -T 'text/plain;' -p big.txt http://localhost:8080/upload
```
